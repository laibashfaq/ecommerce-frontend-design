# Internship_ Week01 Tasks
This project is the Week 1 task for the frontend internship program. The goal was to implement the **header and footer layout** of an e-commerce website based on a provided Figma design using **HTML** and **CSS**.

# 🎯 Task Objectives

- Analyze the given Figma design
- Understand layout structure and visual alignment
- Recreate the **header** and **footer** components
- Practice semantic HTML, flexbox layout, and accessibility

## 📋 Task Requirements

✔️ Extract desktop layout from the Figma file  
✔️ Set up proper folder structure:
- `index.html`
- `CSS/Style.css`
- `images/` for assets

✔️ Implement the following:

### 🔹 Header
- Brand logo
- Search bar with dropdown
- Navigation menu items
- User panel (Profile, Message, Orders, My Cart)
- Language and shipping settings

### 🔹 Footer
- Company branding and description
- Social media icons
- About / Information / Partnership / User links
- App Store & Play Store badges
- Footer bottom section (© copyright)

✔️ Match Figma layout:
- Accurate spacing & alignment
- Font sizes and colors
- Icon usage and hover states

✔️ Push code to GitHub repository
